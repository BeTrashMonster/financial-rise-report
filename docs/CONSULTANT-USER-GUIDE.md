# Financial RISE - Consultant User Guide

**Version:** 1.0
**Last Updated:** 2025-12-22

## Welcome to Financial RISE!

Financial RISE helps you assess your clients' financial readiness and create personalized action plans based on their DISC profile and current phase.

## Getting Started

### 1. Create Your Account
1. Visit [app.financialrise.com](https://app.financialrise.com)
2. Click "Sign Up"
3. Enter your email and create password
4. Verify email address
5. Complete your consultant profile

### 2. Dashboard Overview
**Your dashboard shows:**
- Active assessments
- Completed assessments
- Recent activity
- Quick actions (Create Assessment, View Reports)

## Creating an Assessment

### Step 1: Start New Assessment
1. Click "New Assessment" button
2. Enter client information:
   - Client name
   - Client email
   - Business name
   - Industry (optional)

### Step 2: Choose Assessment Method

**Option A: Self-Administered**
- Client completes independently
- You receive notification when done
- Best for: Tech-savvy clients, remote work

**Option B: Collaborative Session**
- Complete together in real-time
- You ask questions, client responds
- Add notes as you go
- Best for: In-person meetings, hands-on clients

### Step 3: Send to Client (Self-Administered)
- System emails unique link to client
- Client completes at their own pace
- Auto-save protects progress
- You see real-time completion status

## During Assessment

### Collaborative Mode Features
- **Progress Bar:** Shows completion percentage
- **Question Navigator:** Jump to specific questions
- **Consultant Notes:** Add private notes per question
- **Save Draft:** Pause and resume later
- **Client View:** Toggle to see what client sees

### Tips for Success
- ✅ Allow 30-45 minutes uninterrupted
- ✅ Encourage honest responses
- ✅ Explain there are no "wrong" answers
- ✅ Save frequently if collaborative
- ✅ Watch for DISC behavioral cues

## Understanding Results

### The 5 Financial Phases

**Phase 1: STABILIZE** (Foundation)
- Basic accounting health
- Compliance and cleanup
- Debt management
- **Typical Duration:** 3-6 months

**Phase 2: ORGANIZE** (Systems)
- Chart of Accounts setup
- System integration
- Inventory management
- **Typical Duration:** 3-6 months

**Phase 3: BUILD** (Processes)
- SOPs and workflows
- Team financial literacy
- Custom tools/dashboards
- **Typical Duration:** 6-12 months

**Phase 4: GROW** (Planning)
- Cash flow projections
- Revenue forecasting
- Scenario planning
- **Typical Duration:** 6-12 months

**SYSTEMIC** (Cross-cutting)
- Financial literacy
- Report interpretation
- Data-driven decisions
- **Ongoing**

### DISC Profiles

**D (Dominance)** - Results-Focused
- Wants: Quick wins, ROI, bottom line
- Communication: Brief, direct, action-oriented
- Report Style: Bullet points, metrics, fast-paced

**I (Influence)** - Relationship-Focused
- Wants: Collaboration, big picture, opportunities
- Communication: Engaging, story-based, positive
- Report Style: Visual, inspirational, people-centric

**S (Steadiness)** - Process-Focused
- Wants: Step-by-step plan, reassurance, support
- Communication: Patient, detailed, methodical
- Report Style: Sequential, timelines, supportive

**C (Compliance)** - Detail-Focused
- Wants: Data, analysis, thoroughness, accuracy
- Communication: Precise, analytical, fact-based
- Report Style: Comprehensive, metrics, documentation

## Generating Reports

### Consultant Report (Internal Use)
**Contains:**
- Full DISC profile breakdown
- Detailed phase scoring
- All assessment responses
- Consultant notes
- Implementation recommendations
- Red flags and opportunities

**Use For:**
- Your internal analysis
- Planning client presentation
- Creating action plan
- Tracking progress over time

### Client Report (For Presentation)
**Contains:**
- Executive summary
- Current phase explanation
- DISC-adapted action items
- Recommended next steps
- Success metrics
- Resources and support

**DISC-Adapted Language:**
- Automatically tailored to client's profile
- Different tone/detail level per profile
- Optimized for client engagement

### Downloading Reports
1. Navigate to completed assessment
2. Click "Generate Report"
3. Select "Consultant" or "Client"
4. Wait 3-5 seconds for generation
5. Click "Download PDF"

## Presenting Results to Clients

### Match Presentation to DISC Profile

**For D-Profile Clients:**
```
"You're in the BUILD phase. Here are 3 actions that will directly impact your ROI:
1. [Action] - Expected impact: [Metric]
2. [Action] - Expected impact: [Metric]
3. [Action] - Expected impact: [Metric]

Timeline: 90 days. Let's prioritize."
```

**For I-Profile Clients:**
```
"Great news! Your assessment shows exciting opportunities to strengthen your business.
You're in the BUILD phase, which means we can create systems that free up your time
for the parts of the business you love. I've worked with similar businesses that saw
amazing transformations. Let's explore what's possible..."
```

**For S-Profile Clients:**
```
"You're doing well! Your business is in the BUILD phase. I want to reassure you that
we'll take this step-by-step. Here's the plan:

Step 1 (Weeks 1-4): [Action]
Step 2 (Weeks 5-8): [Action]
Step 3 (Weeks 9-12): [Action]

I'll be with you every step, and we'll adjust as needed."
```

**For C-Profile Clients:**
```
"Based on your responses across 5 categories with 25 data points, your business scores:
- Stabilize: 45%
- Organize: 70%
- Build: 85% ← Primary Phase
- Grow: 40%

This data indicates you're in the BUILD phase because [detailed explanation].
Here's the complete breakdown..."
```

## Troubleshooting

**Q: Client didn't receive assessment email**
A: Check spam folder, resend from dashboard, or use direct link

**Q: Assessment won't submit**
A: Ensure all required questions answered, check internet connection

**Q: Report generation failed**
A: Refresh page, try again, contact support if persists

**Q: DISC profile seems inaccurate**
A: DISC is based on assessment responses; discuss with client, consider behavioral observation

**Q: Client in multiple phases**
A: Common during transitions; address both phases in action plan

## Best Practices

✅ **Do:**
- Explain assessment purpose upfront
- Allow uninterrupted time
- Use DISC insights in presentation
- Create collaborative action plans
- Follow up regularly
- Celebrate progress

❌ **Don't:**
- Influence client responses
- Use judgmental language
- Overwhelm with too many actions
- Disappear after assessment
- Compare to other clients
- Make unrealistic promises
